# Criscktech

Projeto estático em HTML para publicação no GitHub.

## Link público

Depois de ativar o GitHub Pages no repositório, o site ficará disponível em:

`https://lucaxcomx0-eng.github.io/criscktech/`

## Como atualizar o projeto

Sempre que você modificar os arquivos, use estes comandos dentro da pasta do projeto:

```powershell
git add .
git commit -m "descreva o que mudou"
git push
```

Exemplo:

```powershell
git add .
git commit -m "atualiza textos da pagina inicial"
git push
```

Após o `git push`, o GitHub recebe as alterações. Se o GitHub Pages estiver ativo, o site público costuma atualizar em alguns minutos.
